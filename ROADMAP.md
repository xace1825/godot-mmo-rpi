# Fantasy Settlement MMO — Roadmap

Цель: сделать persistent-world settlement builder в духе Songs of Syx / RimWorld с MMO-элементами.

---

## Этап 0. Стабилизация (1–3 дня)

Готовность: Windows-клиент подключается и работает без явных багов.

- [x] Сервер на RPi, UDP 7777
- [x] Авто-реконнект клиента
- [x] Ресурсы стартуют на складе (500/500/500)
- [x] Ручной спавн жителей через SPAWN
- [x] Сброс мира через RESET
- [x] Лут виден на клетках (ground items)
- [x] Инфо-панели для складов и жителей
- [x] A* pathfinding
- [ ] Проверить камеру и сквозные кнопки в Windows-клиенте
- [ ] Дотестировать комнаты + indoor-бонус
- [ ] Автосохранение и автозапуск сервера через systemd

Ключевые файлы: `main.gd`, `build_ui.gd`, `network.gd`, `game_state.gd`

---

## Этап 1. Playable Core (1–2 недели)

Готовность: первые 10 минут игры понятны и веселы.

- [ ] Профессии: дровосек, каменщик, фермер, строитель
- [ ] Производственная цепочка: дерево → доски, камень → блоки, еда → еда
- [ ] Жители переносят ресурсы в склад
- [ ] Стены / полы / двери строятся удобно
- [ ] Базовые нужды жителей: еда и сон
- [ ] Сохранение/загрузка мира без потери состояния
- [ ] Визуально читаемая карта и UI

Ключевые файлы: `job_manager.gd`, `game_state.gd`, `planet_generator.gd`, `build_ui.gd`

---

## Этап 2. Depth — Songs of Syx / RimWorld feel (2–3 недели)

Готовность: игра ощущается как полноценный settlement builder.

- [ ] Комнаты и indoor-бонус работают стабильно
- [ ] Жилье и комнаты влияют на настроение/скорость
- [ ] События: набеги, торговцы, погода
- [ ] Первая оборона: враги ломают двери, жители убегают
- [ ] Больше зданий: кухня, кровати, кузница
- [ ] Расширенные инфо-панели

Ключевые файлы: `game_state.gd` (rooms), новые сцены зданий, новый `event_manager.gd`

---

## Этап 3. MMO (2–3 недели)

Готовность: несколько игроков на одном мире.

- [ ] Несколько игроков видят друг друга
- [ ] Совместное строительство
- [ ] Территории или кланы
- [ ] Чат и координаты на карте
- [ ] Сервер сохраняет мир между сессиями

Ключевые файлы: `network.gd`, `game_state.gd`, новый `player_manager.gd`

---

## Этап 4. Content Loop (постоянно)

- [ ] Новые биомы и технологии
- [ ] Больше ассетов DawnLike, анимаций, звуков
- [ ] Баланс, сложность, развитие поселения
- [ ] Возможность присоединиться к публичному серверу

---

## Текущий фокус

Этап 1 — playable core. Базовая стабилизация пройдена; сейчас делаем профессии, производственные цепочки и удобство строительства.

## Конвенции проекта

- `TILE_SIZE = 32`, DawnLike 16→32 nearest
- Авторитет сервера на всем состоянии
- Ручной спавн жителей через SPAWN
- `Engine.time_scale = 5.0` только для тестирования на RPi
- Все ресурсы живут на складах, не в глобальном пуле
- `.godot/` в `.gitignore`, клиент перегенерирует `.ctex`
